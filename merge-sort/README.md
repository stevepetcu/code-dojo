# The Merge Sort

## Problem description

**Given**: A randomly sorted array/list.
**Requirements**: Output the same list, but sorted, using a Merge Sort algorithm.

# Example

Given: `10 3 5 -1 22 11 -3`
Expected result: `-3 -1 3 5 10 11 22
